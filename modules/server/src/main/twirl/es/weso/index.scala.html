RDFShape: RDF playground

Although the main goal of this online demo is to validate RDF data using either SHACL (Shapes Constraint Language) and ShEx (Shape Expressions), it can also be used to

  • Perform SPARQL queries on RDF graphs
  • Perform simple forms of reasoning on top of RDF graphs

More info about the validation languages: Validating RDF Data Book
